# Support / Help

How to get help when you encounter issues with the Payment Services API.

## What to include in a support request

Always include:

- **`paymentId`** — the transaction identifier returned by the API
- **`requestId`** — the UUID you supplied on the request (if used)
- **HTTP status code** — the response code you received
- **Error message** — the content of `errors.message` from the response body
- **Timestamp** — approximate time of the failing request (with timezone)
