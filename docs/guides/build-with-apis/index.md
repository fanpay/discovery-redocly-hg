# Build with APIs

Everything you need to integrate payments and manage customers — from your first API call to production.

## Where do you want to start?

- [Accept Payments](./accept-payments/index.md) — Process card and ACH transactions
- [Manage Customers](./manage-customers/index.md) — Create and update customer profiles

## Frequently asked questions

### Where do I get my API credentials?

See [Get API Key](../global/get-api-key.md) for credential types, where to find them, and how to scope them per environment.

### Which environment should I start with?

Always start in the [Sandbox](../global/sandbox.md). Sandbox credentials only work against the sandbox base URL and no real money moves.

### Is there an OpenAPI spec I can import into my tools?

Yes — the full spec is available at [/apis/openapi.yaml](/apis/openapi.yaml). It is importable into Postman, Insomnia, and any OpenAPI-compatible tool.
